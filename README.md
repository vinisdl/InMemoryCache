# In Memory Cache


Web API Rest with asp net core 2.2 for in memory cache.

Use Swagger for testing.

## Features
1. SET key value
2. SET key value EX seconds (need not implement other SET options)
3. GET key
4. DEL key
5. DBSIZE
6. INCR key
7. ZADD key score member
8. ZCARD key
9. ZRANK key member
10. ZRANGE key start stop
